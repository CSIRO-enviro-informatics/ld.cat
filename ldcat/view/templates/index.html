loci.cat

This loci.cat website is the Loc-I project's Linked Data asset catalogue which can be accessed both by humans and by automated Linked Data tools.

For Loc-I project information, see the Loc-I Project website. For technical, legal and contact information for this tool, see the About page.

Register of Registers

In addition to this page being a basic catalogue homepage, this page is also a Register of Registers, that is it lists all the other registers (lists of items) contained in the catalogue.

The registers this catalogue contains are for:

  1. Datasets
  2. Linksets
  3. Definitional Items
  4. Tools

Linked Data access

This catalogue is available as a Linked Data set of resources. This means it is able to be accessed according to defined information models and in a series of different formats for human and machine-readable purposes.

To view this page's information in according to defined information models and in other formats, see the Alternates View of it.
