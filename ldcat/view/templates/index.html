Linked Data access

This catalogue itself is available as a Linked Data set of resources. This means it is able to be accessed according to Linked Data-defined information models (ontologies) in a series of different formats for human and machine-readable purposes.

To view this page's information in according to defined information models and in other formats, see the Alternates View of it.
